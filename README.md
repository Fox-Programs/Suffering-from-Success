# 🌑 Apprendre à Mourir

> *"Jadis, l'éternité était un don divin. Jusqu’à ce que l'Ombre s’abatte sur l’un d'entre nous."*

Ce projet a été réalisé dans le cadre d'une **Game Jam de 3 semaines**. Le défi majeur ? Découvrir et maîtriser le moteur **Godot 4** (imposé pour l'événement) tout en créant une expérience de jeu complète.

---

## 📜 L'Histoire

Béni par la vie, mais porteur de mort, votre existence dévore celle des autres. Ta famille, tes amis, ton monde... tous s'éteignent pour que tu puisses durer.

**Le paradoxe est cruel :** pour sauver ce qu’il reste, il ne vous reste qu'une seule issue... **apprendre à mourir.**

---

## 🎮 Gameplay & Mécaniques

Dans ce jeu d'action-stratégie, vous devez naviguer dans un monde hostile où votre survie est un fardeau.

* **Gestion du Sacrifice :** Activez des **Totems de Debuff** pour affaiblir votre immortalité dévorante.
* **Survie Tactique :** Gérez les vagues d'ennemis, leurs capacités uniques et vos propres pouvoirs.
* **Équilibre & Chance :** Adaptez votre stratégie en fonction des effets aléatoires et des choix qui s'offrent à vous.

### Commandes
| Action | Touches |
| :--- | :--- |
| **Se déplacer** | `ZQSD` ou `Flèches` |
| **Interagir / Totem** | `E` ou `Entrée` |

---

## 🛠️ Coulisses Techniques (Apprentissage de Godot)

Ce projet fut une immersion totale dans l'écosystème Godot. Voici les concepts clés explorés :

* **Physique 2D :** Transition des `Area2D` vers les `CharacterBody2D` pour un contrôle fluide.
* **Système de Signaux :** Utilisation de signaux globaux pour la communication entre le joueur et l'UI.
* **Animation par code :** Maîtrise des **Tweens** pour les effets visuels (explosions évolutives).
* **Gestion Audio :** Mise en place d'un système de mixage via des **Bus Audio** (Master/Music/SFX).

## 🚀 Installation & Lancement

1. **Téléchargez** le dépôt (ou clonez-le).
2. Ouvrez **Godot Engine 4.x**.
3. **Importez** le fichier `project.godot` situé à la racine.
4. Appuyez sur **F5** pour lancer l'aventure !
